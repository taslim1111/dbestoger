✅ 1. Scroll otomatis ke bawah saat membuka halaman (React)

import { useEffect } from "react";

Tempatkan ini di dalam komponen Kontak.

✅ 2. Scroll ke elemen tertentu (dengan ref)

Jika kamu ingin scroll ke bagian tertentu di halaman (bukan selalu ke bawah), kamu bisa pakai ref:

jsx
import { useRef, useEffect } from "react";

const Kontak = () => {
  const kontakRef = useRef(null);

  useEffect(() => {
    kontakRef.current?.scrollIntoView({ behavior: "smooth" });
  }, []);

  return (
    <section ref={kontakRef} className="Kontak">
      {/* Kontenmu di sini */}
    </section>
  );
};

✅ 3. Scroll saat klik tombol/menu (misalnya dari Navbar)

Di komponen lain (misal Navbar), kamu bisa pakai anchor tag:

jsx

</section>

jsx

✅ Cara Mengubah Ukuran Gambar dalam CSS:

  height: auto;

Penjelasan:
